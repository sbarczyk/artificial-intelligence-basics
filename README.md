# Podstawy Sztucznej Inteligencji

Repozytorium zawiera zestaw laboratoriów z przedmiotu **Podstawy Sztucznej Inteligencji (PSI)**, realizowanego w **obecnym semestrze (V, 2024/2025)** na **Akademii Górniczo-Hutniczej w Krakowie**, na **Wydziale Informatyki**.  
Projekt jest **aktywnie rozwijany**, a kolejne laboratoria dodawane są wraz z postępem zajęć.  
Celem kursu jest praktyczne poznanie i implementacja kluczowych technik uczenia maszynowego oraz metod analizy danych z wykorzystaniem języka **Python** i bibliotek takich jak **NumPy**, **Pandas**, **Scikit-learn** czy **TensorFlow**.
